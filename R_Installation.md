* 講義內容會持續更新, 歡迎follow, fork, pull-request

### GitHub Desktop

[GitHub Desktop](https://desktop.github.com/)

### 設定步驟

* 開啟Git Shell
* 告訴Git你的名字

```
git config --global user.name "YOUR NAME"
```

* 告訴Git你的電子信箱

```
git config --global user.email "YOUR EMAIL ADDRESS"
```
